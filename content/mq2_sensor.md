# 🎤 中級編：センサーで動かそう

ロボットカー虎の巻 その②

*マイクと超音波センサーを使おう — micro:Maqueen*

---

# このミッションでできること

:::

- **マイク（拍手）** でロボットカーをコントロールする
- **超音波センサー** で、ぶつかる前に自動で曲がる（衝突回避）

入門編で走らせたロボットカーに、**まわりを感じる力**をつけよう！

:::

# マイクを見つけよう

:::

micro:bit には **マイク** がついていて、まわりの音の大きさを調べられるよ。

micro:bit の裏側を **よーく見る**と、`MICROPHONE` と書いてあるところがマイクだよ。

:::

![裏側の MICROPHONE がマイク](assets/img/mq2/s06_12.png)

---

# 拍手でLEDを変えよう

:::

まずは、拍手をしたときにLEDの絵が変わるようにしてみよう。

- 「**まわりの音がうるさくなったとき**」で拍手をキャッチ
- 「**しきい値**」を変えて、拍手のときだけ反応するように調整

> しきい値が小さすぎると小さな音でも反応、大きすぎると拍手でも反応しないよ。ちょうどいい数字をさがそう。

:::

![音でアイコンを変えるプログラム](assets/img/mq2/s06_11.png)

---

# 拍手で前に進もう

:::

さっきのプログラムに、走る・止まるブロックを足すよ。

**作ってみよう：**
- 最初は止まっている
- 1回拍手すると、ちょっと前に進む
- もう1回拍手すると、もうちょっと前に進む

:::

![拍手で前に進むプログラム例](assets/img/mq2/s08_16.png)

---

# 拍手で曲がろう

:::

**作ってみよう：**
- 最初はまっすぐ走っている
- 拍手をするたびに、ちょっと右に曲がって、またまっすぐ走る

> 早く走らせすぎると、モーターの音でずっと「うるさい」状態になってしまうよ。速さはひかえめに。

:::

![拍手で右に曲がるプログラム例](assets/img/mq2/s10_19.png)

---

# 変数で交互に曲がろう

:::

今度は、拍手のたびに **右→左→右→左** と交互に曲がってみよう。

> 次に曲がる向きを **「変数」** を使って覚えておくのがコツだよ。

:::

![変数で向きを覚えるプログラム例](assets/img/mq2/s12_22.png)

---

# 超音波センサーとは？

:::

**超音波センサー**は、超音波を出して、物にぶつかって返ってくるまでの **時間できょりを測る** 道具だよ。

- 右（**T**）から超音波を出して、左（**R**）で受け取る
- ロボットカーのセンサーは **2cm〜450cm** まで測れる

> 音は1秒で約340m進むよ。返ってくるまでの時間がわかれば、きょりがわかるんだ。

:::

![超音波センサー](assets/img/mq2/s14_26.jpg)

---

# 距離でドレミを鳴らそう

:::

きょりに合わせて音の高さを変えると、手をかざして演奏できるよ。

> 💪 センサーの前で手を近づけたり遠ざけたりして、ドレミの歌を演奏してみよう！

:::

![きょりでドレミを鳴らすプログラム](assets/img/mq2/s15_30.png)

---

# ぶつかる前に曲がろう（衝突回避）

:::

**作ってみよう：**
- 最初はまっすぐ走っている
- 何かにぶつかりそうになったら、右に曲がる

> 「ぶつかりそうな距離」は何cmがちょうどいいかな？　曲がる速さが速すぎると一回転してしまうよ。

:::

![衝突回避のプログラム例](assets/img/mq2/s17_34.png)

---

# マイク＋超音波で操縦しよう

:::

2つのセンサーを合体させよう。

**作ってみよう：**
- ふだんはまっすぐ走り、ぶつかりそうになると右に曲がる
- 拍手をすると、左に曲がる

> 💪 うまくできたら中級編クリア！ 他にも楽しい操縦アイディアを考えてみよう。 🎉

:::

![マイクと超音波を組み合わせたプログラム例](assets/img/mq2/s18_35.png)
