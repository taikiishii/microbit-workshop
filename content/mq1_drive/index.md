ロボットカー虎の巻 その①

---

# このミッションでできること

:::

- ロボットカーを **前後** に動かす
- **右折・左折**、その場で **まわれ右・まわれ左**
- **ジグザグ・波型** に走らせる
- **三角形・四角形・円** を描いて走らせる

プログラム次第で、思いどおりに走らせられるようになるよ！

:::

# メイクコードを開こう

:::

プログラムは **メイクコード** で作るよ。

1. ブラウザで `makecode.microbit.org` をひらく
2. 「**新しいプロジェクト**」をクリック
3. 名前をつけて「**作成**」

> 画面が開いたら大成功！ うまくいかないときはメンターによんでね。

:::

![名前をつけて「作成」](img/s06_16.png)

---

# マイクロビットをパソコンにつなごう

:::

1. 左下の「**・・・**」→「**デバイスを接続する**」→「**次へ**」
2. 「**ペア**」→ `BBC micro:bit` をえらんで「**接続**」

> ⚠ エラーが出たらメンターに相談してね。一度つなげば、次からは自動でつながるよ。

:::

![ペアリングの画面](img/s09_23.png)

---

# 最初のプログラム（力だめし）

:::

まずは文字を出してみよう。

1. 「**基本**」→「**文字列を表示**」を「**ずっと**」の中へ
2. むらさきの「**ダウンロード**」で書きこむ
3. マイクロビットに文字が流れたら成功！

:::

![「文字列を表示」ブロック](img/s10_27.png)
![マイクロビットに表示された](img/s10_31.png)

# ロボットカーをよく見よう

:::

ロボットカーにはたくさんの部品がついているよ。

- 前の **超音波センサー**（きょりを測る）
- 左右の **モーター**（タイヤを回す）
- 裏の **ラインセンサー**（線を見る）
- **フルカラーLED**・**スピーカー**・**電源スイッチ**

:::

![micro:Maqueen をま上から見たところ](img/s13_36.png)

---

# ロボットカーを組み立てよう

:::

1. **電池**を入れる（＋−の向きに注意！）
2. **マイクロビット** をさしこむ（向きに注意！）
3. **超音波センサー** をさす

> タイヤや電池ボックスがまだなら、説明書を見ながら組み立てよう。組めたら電源スイッチはまだ **OFF** のままにしておこう。

:::

![組み立てたロボットカー](img/s14_40.png)

---

# 拡張機能を追加しよう

:::

ロボットカーを動かすブロックを使えるようにするよ。

1. 「**拡張機能**」をクリック
2. `maqueen` をさがして、出てきたカードをえらぶ
3. 「**Maqueen**」と「**IR**」のメニューがふえる

> IR は「赤外線（せきがいせん）」という意味だよ。

:::

![この「maqueen」をえらぶ](img/s15_43.png)

---

# 増えた機能を見てみよう

「**Maqueen**」のメニューに、モーターやLED、センサーのブロックがふえたよ。

どんな使い方ができるか、想像できるかな？ これから少しずつ使っていこう。

:::

# タイヤを動かしてみよう

:::

「すべてのモーターを 前へ 速さ 255 で回す」ブロックで走らせてみよう。

> ⚠ スイッチ **OFF** のまま書きこんで、安全で広いところに運んでから **ON** にしよう。

> 💪 やってみよう！　もっとゆっくり走らせるには？　後ろ向きに走らせるには？　左右のモーターを逆向きにするとどうなる？

:::

![すべてのモーターを前に回す](img/s17_51.png)

---

# タイマーと組み合わせよう

:::

「**一時停止**」を使うと、時間で動きを区切れるよ。

このプログラムは「**1秒 前に進んでから止まる**」だよ。

> モーターは、速さを変えるか止めるまで、同じ速さで回り続けるよ。

> 💪 やってみよう！　「1秒 前に進んで、1秒 後ろに進む」をくり返すには？

:::

![1秒前進して止まる](img/s18_58.png)

---

# 右折・左折してみよう

:::

**左のモーターだけ**を前に回して、**右のモーターを止める**と、右に曲がるよ。

> 💪 やってみよう！　ちょうど直角に曲がるには、どの数字を調整すればいいかな？　右のモーターを止めずにゆっくり回すと、どんな動きになる？

:::

![1秒前進して右に曲がる](img/s19_62.png)

---

# 三角形・四角形・円を描こう

:::

「まっすぐ進む」と「曲がる」を組み合わせると、いろいろな形が描けるよ。

**曲がる速さ**と**曲がる時間**を調整してみよう。

> 💪 三角形・四角形・円を、それぞれ描けるかな？　うまくできたら入門編クリア！ 🎉

:::

![進む＋曲がるで形を描く](img/s20_63.png)
