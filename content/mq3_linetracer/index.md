# ライントレーサーとは？

:::

**ライントレーサー**は、床に描かれた線の上を **自動で進んでいく** ロボットカーだよ。線が曲がっていても、ちゃんと線に沿って動くんだ。

ロボットカーには、床の色（**白か黒**）を見分けられるセンサーがついていて、「いま線の上にいるかどうか」を調べられるよ。

:::

![線をたどって自動で走る](img/s06_10.png)

---

# ライントレーサーの仕組み

:::

ロボットカーには **2つのセンサー** がついていて、線に対して「真上・右にずれた・左にずれた」がわかるよ。これをくり返して、線から外れないように進むんだ。

- 🟢 線の **真上** → そのまま **まっすぐ** 進む
- ↩️ 線の **右** にずれた → **左** に曲がって線にもどる
- ↪️ 線の **左** にずれた → **右** に曲がって線にもどる

:::

![線をたどるロボットカー](img/s06_13.png)

---

# センサーを見てみよう

:::

ロボットカーの **裏側** を見ると、**ラインセンサーが2つ** ついているよ。

それぞれに、赤外線を出す **LED** と、赤外線の量を測る **センサー** があるんだ。

- 真下が **白い床** → 赤外線がたくさん反射して返ってくる
- 真下が **黒いライン** → 赤外線が少ししか返ってこない

この差で、白か黒かを見分けているよ。

:::

![ロボットカーのセンサー](img/s09_18.png)

---

# センサーの値を調べよう

:::

まずは、センサーが返す **数字（値）** を自分の目で確かめよう。

1. 「**ラインセンサーの値を数で表示**」するプログラムを作る
2. ロボットカーを **床の上** に置いたときの値をメモ
3. **ラインの上** に置いたときの値をメモ

> 右センサー・左センサーそれぞれで、白のときと黒のときの値をくらべてみよう。この差がプログラムのカギになるよ。

:::

![左右のラインセンサーの値を表示](img/s10_21.png)
![右のラインセンサーの値を表示](img/s10_23.png)

# ライントレースのプログラムを作ろう

:::

「**線の上**」「**線の右にずれた**」「**線の左にずれた**」の3つの場合を考えて、それぞれの動きを決めよう。

> ⚠ 速く動かしすぎると、ラインから外れてしまうよ。ゆっくりから始めよう。

> 💪 3つの場合を「もし〜なら」でプログラムにできるかな？

:::

![ライントレースのプログラム例](img/s11_25.png)

# エッジトレーサーを作ろう

:::

**エッジトレーサー**は、白と黒の **境界（エッジ）** をたどる方法だよ。ラインが太くても、うまく動くのがいいところ。

- 🟢 **境界の上** → まっすぐ進む
- ↩️ **白い方** にはみ出した → 少し曲がってもどる
- ↪️ **黒い方** に入りすぎた → 少し曲がってもどる

> 💪 うまくできたら上級編クリア！ さらに超音波センサーを足して、障害物があったら止まるようにできるかな？ 🎉

:::
