# 揺すって音を鳴らそう

---

# なにをするの？

:::

マイクロビットには **スピーカー** があって、音や音楽を鳴らせるよ。

さらに **加速度センサー** も入っていて、かたむきを調べたり、**揺らしたとき**にプログラムを動かしたりできるんだ。

今回は「**揺らしたら音が鳴る**」プログラムを作ってみよう！

:::

![スピーカーと加速度センサーが入っているよ](img/s02_02.png)

---

# 揺らしたら音を鳴らそう

:::

1. 「**入力**」のメニューから「**ゆさぶられたとき**」ブロックを置く
2. 「**音楽**」のメニューから「**鳴らす**」ブロックを、その中にドラッグ

> 「最初だけ」と「ずっと」の中は、空っぽにしておこう。

:::

![「入力」メニューのブロック](img/s03_03.png)
![組み合わせたところ](img/s03_06.png)

---

# ためして書きこもう

:::

むらさきの「**ダウンロード**」でマイクロビットに書きこもう。

書きこめたら、マイクロビットを手に持って **ふってみよう**。音が鳴ったら大成功！ 🎉

> 音が鳴らないときは、電池やUSBでちゃんと電源が入っているか確かめてね。

:::

![ふると音が鳴るよ](img/s04_09.png)

:::
