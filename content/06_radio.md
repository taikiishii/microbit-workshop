# 📡 無線でつうしんしよう

マイクロビット どうしでメッセージを送り合おう

*マイクロビット 体験ワークショップ*

---

# なにをするの？

:::

マイクロビットには **無線（むせん）** の機能があって、**2台以上のマイクロビット** で電波を使ってメッセージを送り合えるよ。

- 送る側と受け取る側の **グループ番号を同じ**にすると、お話しできる
- ちがうグループには聞こえないから、近くでたくさん使っても混ざらないよ

友だちのマイクロビットと一緒にためしてみよう！

:::

![電波でメッセージを送り合う](img/concept.png)

---

# ① グループを合わせよう

:::

まず、話す相手と **同じグループ番号** にそろえるよ。

1. 「**無線**」のメニューから「**無線のグループを設定**」を「**最初だけ**」の中に入れる
2. 番号を決める（れい：`1`）

> ⚠ 送る人と受け取る人、**両方のマイクロビット** に同じ番号を入れてね。番号がちがうと通じないよ。

:::

![最初だけ：無線のグループを設定 1](img/group.png)

---

# ② メッセージを送ろう

:::

ボタンをきっかけに、数字を送ってみよう。

1. 「**入力**」から「**ボタンAが押されたとき**」を置く
2. 「**無線**」から「**無線で数値を送信**」を中に入れる
3. 送る数字を決める（れい：`1`）

:::

![ボタンAが押されたとき：無線で数値を送信 1](img/send.png)

---

# ③ メッセージを受け取ろう

:::

今度は、受け取ったときの動きを決めるよ。

1. 「**無線**」から「**無線で受信したとき**」を置く
2. 中に「**アイコンを表示**」などを入れる

これで、相手がボタンを押すと、自分のマイクロビットにハートが出るよ！

:::

![無線で受信したとき：アイコンを表示](img/receive.png)

---

# 送り合ってみよう

:::

送る・受け取るを合体させると、**ふって（ゆさぶって）送る → 相手の画面に数が出る** ができるよ。

2台のマイクロビットに同じプログラムを書きこんで、ふってみよう。

> 💪 やってみよう！　ボタンAで「1」、ボタンBで「2」を送って、受け取った数で顔を変えられるかな？　じゃんけんゲームも作れるよ。

:::

![ふって送る／受信して数を表示](img/example.png)
